# 🏦 Banking System Website

A simple **Banking System Web Application UI** built using **HTML and CSS**.  
This project simulates the interface of an **online banking system** with both **User and Admin panels**.

The system allows users to perform basic banking operations such as creating an account, transferring money, checking balance, and viewing transaction history.

---

## 🚀 Features

### 👤 User Panel
- User Login
- Create New Bank Account
- Transfer Money
- Credit Amount (Deposit)
- Debit Amount (Withdraw)
- Check Account Balance
- View Transaction History
- Change Password

### 🛠 Admin Panel
- Admin Dashboard
- View Account Details
- View Transaction History
- View Debit History
- View Credit History
- Manage Banking Records
